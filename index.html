HTML ist die Sprache, in der die meisten Websites geschrieben sind . HTML wird verwendet, um Seiten zu erstellen und funktionsfähig zu machen.

Der Code, der verwendet wird, um sie optisch ansprechend zu gestalten, wird als CSS bezeichnet und wir werden uns in einem späteren Tutorial darauf konzentrieren. Im Moment konzentrieren wir uns darauf, Ihnen das Bauen und nicht das Entwerfen beizubringen.

Was sind Tags und Attribute?

Tags und Attribute sind die Grundlage von HTML.

Sie arbeiten zusammen, erfüllen aber unterschiedliche Funktionen – es lohnt sich, zwei Minuten in die Unterscheidung der beiden zu investieren.

Tags werden verwendet, um den Anfang eines HTML-Elements zu kennzeichnen , und werden normalerweise in spitze Klammern eingeschlossen. Ein Beispiel für ein Tag ist:

<a href="https://www.example.com">Link zu Beispiel.com</a>

Erstellen Ihrer ersten HTML-Webseite

Zunächst müssen Sie Ihren HTML-Editor öffnen. Dort finden Sie eine saubere weiße Seite , auf der Sie Ihren Code schreiben können.

Von dort aus müssen Sie Ihre Seite mit den folgenden Tags gestalten.

Grundlegender Aufbau einer HTML-Seite:

Diese Tags sollten untereinander oben auf jeder von Ihnen erstellten HTML-Seite platziert werden.

  • !DOCTYPE html– Dieses Tag gibt die Sprache an, in der Sie auf der Seite schreiben. In diesem Fall ist die Sprache HTML 5

  • html— Dieses Tag signalisiert, dass wir von nun an im HTML-Code schreiben werden.

  • head– Hier werden alle Metadaten der Seite abgelegt – Dinge, die hauptsächlich für Suchmaschinen und andere Computerprogramme gedacht sind.

  • body— Hierhin geht der Inhalt der Seite .
<!DOCTYPE html> <html> <head> <title>Titel Ihrer Webseite</title> </head> <body> <h1>Willkommen auf meiner Webseite</h1> <p>Hier ist etwas Text.</p> </body> </html>
Inhalte hinzufügen

Als nächstes erstellen wir "body" ein Tag. Im HTML-Code "body" fügen wir den Inhalt hinzu, der für die Betrachtung durch das menschliche Auge konzipiert ist. Dazu gehören Texte, Bilder, Tabellen, Formulare und alles andere, was wir täglich im Internet sehen.

Das Hinzufügen von Text zu unserer HTML-Seite ist einfach, indem Sie ein mit dem Tag geöffnetes Element verwenden, "p" das einen neuen Absatz erstellt . Wir platzieren unseren gesamten regulären Text innerhalb des Elements "p".

Wie Sie vielleicht bemerkt haben, besteht das Internet aus vielen Links.Links werden in ein Attribut eingefügt, das durch das "a" -Tag geöffnet wird. Dieses Element ist das erste, das wir kennengelernt haben, das ein Attribut verwendet und daher anders aussieht als die zuvor erwähnten Tags . In der modernen digitalen Welt von heute sind Bilder alles. Das "img" -Tag enthält alles, was Sie zum Anzeigen von Bildern auf Ihrer Website benötigen. Ähnlich wie das Ankerelement "a" enthält auch "img" ein Attribut.
